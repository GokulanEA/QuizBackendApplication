QuizBackendApplication is a Spring Boot-based REST API designed to manage quizzes and questions efficiently. The application enables dynamic quiz creation by selecting a specific category and number of questions, and it ensures secure data handling by returning questions without exposing correct answers through the use of a DTO (Data Transfer Object) pattern. It follows a layered architecture consisting of Controller, Service, Repository, and Entity layers, promoting clean code organization and maintainability. The backend is integrated with a MySQL database using Spring Data JPA and Hibernate, and it also includes custom query methods for random question selection. All APIs have been thoroughly tested using Postman to verify request-response behavior and ensure proper backend functionality. This project demonstrates key backend development concepts such as RESTful API design, entity relationships, dependency injection, and database integration, making it suitable for learning as well as real-world application development.
